# Lucien - Emotional AI (v1)

This is my first project as I transition into Artificial Intelligence.

Lucien is a simple emotional detection program that identifies basic emotions from user input using keyword matching.

## Features
- Detects emotions like sadness, anger, fear, and happiness
- Basic input validation
- First step toward an empathetic AI system for children (for now)

## Next Steps
- Improve emotion detection with NLP
- Add more emotions
- Build an interactive interface

---

This is version 1 of a bigger vision: creating AI that understands and supports human emotions.
